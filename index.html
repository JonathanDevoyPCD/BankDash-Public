Updates Log
Features, fixes, and improvements shipped so far
v0.11.0June 2026
Public signup and onboarding
  • Added a create-account flow for new BankDash users.
  • Added email confirmation messaging for Supabase signup.
  • Added first-login account setup routing so new users can configure profile and dashboard defaults.
v0.10.0June 2026
Public layout cleanup
  • Removed duplicate public dashboard sections that were no longer useful.
  • Removed Budget Plan and Priority Rules from the public navigation and page flow.
  • Consolidated Statements, Payments, and Transactions into a future Bank Activity workflow.
  • Parked Bank Activity from the public UI until statement uploads and transaction review are ready for release.
v0.9.0June 2026
Shopping List budgeting
  • Added the Shopping List modal for grocery planning.
  • Shopping List can replace the Groceries checklist item when enabled.
  • Added paid/pending shopping items, max budget, totals, and synced shopping state.
  • Added PayJustNow as a checklist item.
v0.8.0June 2026
Account data sync
  • Added Supabase-backed user app state sync across browsers and PCs.
  • Added Sync Account and Force New Sync behavior.
  • BankDash now checks for newer synced data after login.
v0.7.0May 2026
Banking and card tools
  • Added primary bank and virtual card panels.
  • Added pre-loaded bank selection with bank colours and logos.
  • Added password-gated card detail reveal for the current session.
v0.6.0May 2026
Manual financial planning
  • Converted Overview into a manual current-cycle dashboard.
  • Added editable salary, expenses paid, GoalSave, and total left over blocks.
  • Expanded Priority Checklist with max spend, spent amount, status, active/inactive controls, and budget states.
v0.5.0May 2026
Cleaner dashboard layout
  • Slimmed down duplicate dashboard sections.
  • Kept unique sections focused on clear functions, useful summaries, and action alerts.
  • Added sidebar controls, calculator modal, privacy toggle improvements, and status colours.
v0.4.0May 2026
PDF upload and review import
  • Added PDF upload and statement parsing for TymeBank statements.
  • Added review imported transactions before final save.
  • Improved parser accuracy, fee matching, EveryDay account filtering, and bulk upload limits.
v0.3.0May 2026
Supabase account foundation
  • Added Supabase auth, password reset, protected dashboard access, and account settings.
  • Added database tables for profiles, transactions, categories, uploads, recurring payments, budgets, and goals.
  • Added toast notifications for save, load, warning, and error feedback.
v0.2.0May 2026
Finance insights and rules
  • Added 25-to-24 cycle logic for month-to-month finance tracking.
  • Added essential payments, Google payment allowlist, recurring payments, and estimated next-month calculations.
  • Added expandable transaction breakdowns for priority and non-priority sections.
v0.1.0May 2026
Initial BankDash dashboard
  • Created the first financial dashboard from bank statement data.
  • Added income, expenses, priority payments, non-priority payments, and net savings summaries.
  • Added date filters, privacy blur, and initial dashboard styling from the template.
